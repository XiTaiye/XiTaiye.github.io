  • HTML(Hyper Text Markup Language): 超文本标记语言

04. 标签

1. 注释

  • 快捷键 ctrl + /

2. 标题标签

  • html一共提供h1 到 h6，6种标题

  • ctrl + d一次多一个光标

3. 段落标签

  • paragraph

  • 当段落的内容很多时，自动换行功能

4. 换行标签(单标签)

  • break

5. 水平尺标签（单标签）

  • Horizontal Rule

6. 文本格式化标签

  • b: bold
  • u: underline
  • i: italic
  • s: strikethrough
  • strong: strong
  • ins: inserted
  • em: emphasize
  • del: delete

7. 图片标签

  • img : image
  • alt: alter

  • 一个标签可以有多个属性，属性之间用空格隔开，空格。

  • width和height只写一个就行，浏览器会等比例缩放

绝对路径：从盘符出发，文件的绝对位置

  • 网址也是绝对路径
