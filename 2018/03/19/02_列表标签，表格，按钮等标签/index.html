02_列表标签，表格，按钮等标签

01. 列表标签

1. 无序列表

  • ul: unordered list
  • li: list item

  • 列表的每一项前默认显示圆点
  • ul标签中只允许包含li标签
  • li标签可以包含任意内容

2. 有序列表

  • ol： ordered list

  • li： list item

3. 自定义列表

  • dl： definition list
  • dt： definition term
  • dd：definition description

  • dl标签只允许包含dt/dd标签
  • dt/dd标签可以包含任意内容

02. 表格

1. 标签

  • table
  • tr： table roll
  • td： table data cell
  • border
  • width
  • height

2.表格标题和表头单元格标签

  • caption
  • th： table header cell

3.表格的结构标签

  • thead
  • tbody
  • tfoot
    • 表格结构标签内存用于包裹tr标签
    • 表格的结构标签可以省略

4. 合并单元格

  • rowspan: 保留最上的，删除其他的
  • colspan： 保留最左的，删除其他的

  • 同一结构标签中的单元格才能合并，不能跨结构标签合并（不能跨：thead、tbody、tfoot）

03. 表单标签

1. input系列标签

  • radio：A radio button or option button is a type of graphical user interface element that allows the user to choose only one of a predefined set *of options (that’s opposite to tick boxes).

    They were named after the physical buttons used on older* car radios to select preset stations - when one of the buttons was pressed, other buttons would pop out, leaving the pressed button the only button in the “pushed in” position

  • placeholder： 占位符

2. text

  • input不换行

3. password

4. radio

  • 单选框

5. checkbox

  • 多选框

6. file

  • 上传文件

7.placeholder

  • 占位符

8. name

9. checked

10.multiple

  • 上传多个文件

04. 按钮

  • submit
  • reset
  • button
  • 必须包含在表单域中
    标签

05. button按钮标签

06. select标签

  • select
  • option
  • selected

07. 文本域标签

  • texitarea
  • cols
  • rows

08. label标签

  • label： 点字也可以勾选

09. 语义化标签

  • div：一行只显示一个（独占一行）
  • span：一行可以显示多个

  • 这些用在手机网页中
