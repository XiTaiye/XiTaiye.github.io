02_关键字、变量、运算符

01. 关键字

  • 关键字全部小写
  • 编译器中有颜色变化

02. 常量

  • 常量： 在程序的执行过程中，其值不会改变的量

常量的分类

  • 字符串常量
  • 整数常量
  • 小数常量
  • 字符常量：’1’（字符1）
  • 布尔常量
  • 空常量

03. 变量

  • 变量是内存中的存储空间

  • 定义格式 ： 数据类型 变量名 = 数据值;

计算中最小的存储的单元是字节，一个字节有8位

注：基本数据类型中没有字符串

04. 变量的注意事项

  1. 变量名不允许重复定义
  2. 一条语句可以定义多个变量，使用逗号分隔，以分号结束
  3. 使用变量之前要进行赋值
  4. 定义float类型变量时，后边要加f。eg. float a = 12.3f;
  5. 定义long变量变量时，后面要加L。eg. long a = 123L;
  6. 变量的作用域范围，只在他所在的大括号中有效

05. 键盘录入

  1. 导包
  2. 创建对象
  3. 使用变量接收数据

06. 标识符

  • 由数字、字符、下划线和$造成
  • 不能以数字开头
  • 不能是关键字
  • 区分大小写

07. 类型转换

  • 隐式转换：取值范围小的数据可以直接给取值范围大的类型赋值

  • byte, short, char,在运算的时候，都会直接提升到int，然后进行运算

  • 强制转换：取值范围大的数值或者变量赋值给另一个表示数据范围小的数据

  • Java的常量优化机制

08. 运算符

1. 算术运算符

取模和取余是有区别的，当除数和被除数都是正数不会有区别，但是当有负数的时候会有区别。

字符串的拼接

  • 使用加号可以与任意数据类型拼接

2. 自增自减运算符

  • 单独使用并没有区别

3. 赋值运算符

  • 底层自带强转功能

4. 关系运算符

5. 逻辑运算符

  • 异或，相同则false，不同则true

6. 三元运算符

  • 格式：关系表达式 ？ 表达式1(true) ： 表达式2(false)
    • eg. int max = a > b ? a : b;
