01_ 初识Java

01. 快捷键

  • win + V ：打开粘贴板

  • F2 ： 重命名

02. Java跨平台原理

原理：JVM虚拟机（虚拟机本身不能跨平台）

03. （了解）JRE和JDK

JDK：Java Development Kit，包含了编译工具和运行工具

JRE：Java Runtime Environment,包含JVM和Java核心类库

04. JDK的下载和安装

  • 安装目录，再建立一个jdk-12文件夹，不然以后安装多个版本的jdk会很乱

05. 常用DOS命令

  • dir在mac是ls
  • tab键是自动补全

06. Path环境变量的配置

  • 作用：在任意路径下都能访问jdk的bin文件夹
  • 环境变量就是装载入路径的容器

07. HelloWorld案例编写

  • 编译 javac 文件名.java
  • 执行 java 文件名

08. 注释

  • 单行注释

  • 多行注释

  • 文档注释
