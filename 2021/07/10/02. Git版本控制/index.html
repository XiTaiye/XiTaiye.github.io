Git

什么是git：

分布式的版本控制工具，通过Git仓库来存储和管理文件，仓库分为本地仓库和远程仓库

作用：

  • 代码回溯
  • 版本切换
  • 多人协作
  • 远程备份

常用的Git托管服务：

  • gitHub
  • 码云
  • Bitbucket
  • GitLab(需要自己搭建)

Git常用命令

  • 设置 用户信息

  • Git文件的两种状态

    • untracked
    • tracked
      • Unmodified
      • Modified
      • Staged
